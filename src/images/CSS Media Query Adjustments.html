To create a wave-like effect on an image where the image grows based on the angle of the mouse hover, you can use a combination of CSS and JavaScript. The idea is to dynamically apply transformations to the image based on the mouse's position relative to the image. Here's a step-by-step guide:

1. HTML

2. CSS

Set up the basic styles for your image and apply a transition for smooth scaling:

css
#hero-image {
    width: 200px; /* Adjust as needed */
    height: 200px; /* Adjust as needed */
    border-radius: 50%; /* Make it circular */
    transition: transform 0.2s ease-out;
    position: relative;
}

3. JavaScript

Use JavaScript to calculate the angle of the mouse relative to the center of the image and apply a transformation based on that angle.

javascript
document.addEventListener('mousemove', function(e) {
    const image = document.getElementById('hero-image');
    const rect = image.getBoundingClientRect();
    const centerX = rect.left + rect.width / 2;
    const centerY = rect.top + rect.height / 2;

    // Get mouse position
    const mouseX = e.clientX;
    const mouseY = e.clientY;

    // Calculate the angle from the center of the image to the mouse position
    const angle = Math.atan2(mouseY - centerY, mouseX - centerX) * (180 / Math.PI);

    // Calculate scale based on angle, you can adjust this formula to achieve the desired effect
    const scale = Math.abs(Math.sin(angle * Math.PI / 180)) * 0.3 + 1;

    // Apply the transformation
    image.style.transform = `scale(${scale})`;
});

Explanation:

  • CSS: Defines the initial style of the image and ensures that the transition is smooth.
  • JavaScript: Calculates the angle between the mouse position and the center of the image using Math.atan2. The scale is adjusted based on the sine of this angle, which gives a wave-like effect.

Optional Enhancements:

  • Performance: To improve performance, you could debounce the mousemove event.
  • Boundary Checks: Ensure that the image doesn't exceed its container boundaries or use clip-path for more complex effects.

Feel free to tweak the scale formula and transition times to fit your desired wave effect!

Modify the JavaScript to handle the mouseenter and mouseleave events:

const image = document.getElementById('hero-image');

function handleMouseMove(e) {
    const rect = image.getBoundingClientRect();
    const centerX = rect.left + rect.width / 2;
    const centerY = rect.top + rect.height / 2;

    // Get mouse position
    const mouseX = e.clientX;
    const mouseY = e.clientY;

    // Calculate the angle from the center of the image to the mouse position
    const angle = Math.atan2(mouseY - centerY, mouseX - centerX) * (180 / Math.PI);

    // Apply the transformation

function handleMouseEnter() {
    document.addEventListener('mousemove', handleMouseMove);
}

function handleMouseLeave() {
    document.removeEventListener('mousemove', handleMouseMove);
    // Reset the scale when the mouse leaves
    image.style.transform = 'scale(1)';
}

// Add event listeners
image.addEventListener('mouseenter', handleMouseEnter);
image.addEventListener('mouseleave', handleMouseLeave);

Explanation:

  • handleMouseEnter: Adds the mousemove event listener when the mouse enters the image area.
  • handleMouseLeave: Removes the mousemove event listener and resets the image scale when the mouse leaves the image area.
  • handleMouseMove: Applies the scale transformation based on the mouse position relative to the center of the image.

This setup ensures that the image only transforms while the mouse is over it, avoiding unnecessary calculations when the mouse is nearby but not directly hovering over the image.
